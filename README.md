# Описание приложения
Мобильное приложение онлайн магазина по продаже одежды.  
Предоставляет пользователю доступ к каталогу вещей с возможностью фильтрации по типу, материалу, размеру, цене и другим параметрам. Каждое изделие сопровождается подробным описанием, характеристиками, фотографиями высокого качества и информацией о наличии на складе.
## Клонирование проекта
Откройте терминал или Git Bash.

Выполните команду:
```bash
git clone https://github.com/FEIP-FEFU-Mobile-Spring-2026/team-kfc
```
Перейдите в папку проекта:
```bash
cd <название_проекта>
```
## Открытие проекта в Android Studio
1. Запустите Android Studio.
2. Выберите Open.
3. Укажите папку с проектом.
4. Дождитесь завершения синхронизации Gradle (может занять несколько минут).
## Сборка проекта
Сборка выполняется автоматически после синхронизации Gradle.
- Если требуется вручную:
В меню выберите Build → Make Project
- Или через терминал:
```bash
./gradlew build
```
#### Debug сборка
```bash
./gradlew assembleDebug
```
## Запуск приложения
- Через эмулятор:
1. Откройте Device Manager.
2. Создайте или запустите виртуальное устройство (AVD).
3. Нажмите кнопку Run (▶) в Android Studio.
4. Выберите эмулятор.
- Через физическое устройство:
1. Включите Режим разработчика на устройстве.
2. Активируйте USB-отладку.
3. Подключите устройство к компьютеру.
4. Разрешите отладку на устройстве.
5. Нажмите Run (▶) и выберите устройство.
