# Описание приложения
Мобильное приложение онлайн магазина по продаже одежды.  
Предоставляет пользователю доступ к каталогу вещей с возможностью фильтрации по типу, материалу, размеру, цене и другим параметрам. Каждое изделие сопровождается подробным описанием, характеристиками, фотографиями высокого качества и информацией о наличии на складе.
